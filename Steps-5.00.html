Split Columns with multiple column values into single value columns. 5

 

The column WeaponUsed has multiple values in some rows. For example, the row with ShootingID = 47 has “Handgun Rifle Shotgun” as the value for this column. It must be split into 3 rows with the same ShootingId with a unique value for the WeaponUsed column. The data at this point has 118 rows. After this step, the data should have 172 rows.






5.1 – Split multiple values into rows.

1. Select the column with the multiple values. In this case WeaponUsed.

2. Go to the Transform tab.

3. Click on Split Column from the Text Column menu.

4. Select the option By Delimiter.

5. In the new window, make sure that the space delimiter is selected.

6. Also the option Each occurrence of the delimiter.

7. In the advance options tab, select Rows for the Split Into options.

8. Click Ok.






If done correctly, there should be 172 rows. Now for ShootingID = 47, there should be 3 rows, with a unique value in the Weapon Used.
